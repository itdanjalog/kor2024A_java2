1.단어
    - 한국인(한국어) : 문자
        - 코드 집합( JAVA , C , PYTHON 등등 )
        야 컴퓨터야 출력해 -----> 010101111111111
        야 컴퓨터야 출력해 ----> JAVA( System.out.print() ) , PYTHON( print() ) ---> 컴파일(번역)-> 01010101
    - 기계어 : 0 , 1
    - 코드 : 0,1 을 이용한 미리 정해진 문자/문법
    - 컴파일 : 코드을 기계로 번역해주는 행위 , 컴파일러 : 번역 프로그램

    - 프로그램 : 코드/명령 의 집합
    - 프로그래머 : 프로그램을 만드는 사람
    - 프로그래밍 : 프로그램을 만드는 일
    - 프로그래밍 언어 : JAVA , C , PYTHON ,JS  등등

2. 자바 사용하는 이유
    1. [운영체제로 부터 독립적이다]동일한 코드가 모든 운영체제 에서 실행 가능
    2. 객체지향 , 유지보수 , 확장성이 좋다 , 안정적이다 , 메모리 효율성( G 쓰레기 C 수집기 = 사용하지 않는 메모리를 자동 삭제 ) 좋다
    3. 풍부한 라이브러리( 미리 만든 함수/클래스) 제공한다.

객체란? vs 주체란?
    주체(Subject) : '나' 라는 실체
    객체(Object) : 주체(나) 가 아닌 다른 주체(타인)
        즉] 본인 이외의 다른 실체 의미
        - 프로그래머(주체)가 프로그램(pc)내 다른 실체(객체)를 다루는 작업

    '나' 주체 : 이름 , 나이 , 직업 , 연락처 , 걷는스타일 , 입모양
    '타인'객체 : 김현수 40 강사 010-3913-2072 팔자걸음 먹을때큰입
                     이름 , 나이 , 직업 , 연락처 ( 속성 ) 고유성질
                     걷는스타일 , 입모양 ( 메소드 ) 행동
    '타인'객체 : 유재석 42 개그맨 010-3913-2073 팔자걸음 먹을때큰입

변수란 ? 변하는 수 , 자료 1개
    자료형 변수명 = 리터럴 또는 수식 등등
상수란 ? 변하지 않는 수 , 자료 1개
    final 자료형 변수명 = 리터럴 또는 수식 등등
배열이란 ? 여러 자료들을 하나의 자료로 묶음
    자료형[] 변수명 = new 자료형[길이]
    - 인덱스 : 여러 자료들을 한곳에 묶었기 때문에 저장된 순서(식별번호) 부여
        - 0번부터 시작
메소드/함수 란 ? 여러개 코드/명령/실행 문( ; ) 을 묶음

클래스 란 ? 설계도
    왜? 객체( 필드 , 생성자 , 메소드 )을 만들기 위해 미리 만들어두는 틀/설계도 , 서로 다른 객체을 대량 생산
    클래스의 멤버 : 필드(고유성질) , 생성자( new ) , 메소드(행위/행동/방법)

객체/object 란 ? 설계도/타입 기반으로 생성된 메모리 을 포함한 추상적인 개념

인스턴스/instance 란 ? 클래스 기반으로 컴퓨터에 실제로 메모리 할당된 상태

상속 이란 ? extends(연장) , A클래스가 B클래스에게 A클래스의 멤버을 B클래스가 사용할수 있도록 물려주는 행위
    왜? 물려받으면 더 크게 사용할수 있기 때문에 , 클래스로 부터 물려받아 또 다른 클래스를 만들면 빠른 설계 가능하다.
   오버라이딩 : @Override 물려받은 메소드을 나만의 것으로 재정의/커스텀
   오버로딩 : 동일한 메소드명으로 여러개를 선언하는 방법론 , ( 매개변수 개수 와 타입 이 다르면 가능 )
   - 다형성 : 다양한 형태 가지는 성질 , 같은 코드인데 다르게 표현 수정/커스텀/재정의

인터페이스란 ? 자바에서 100% 다형성을 표현 하기 위한 문법 제시

                    ----> 에어컨
    사용자 ---> 리모콘 ----> TV
                    ----> 냉장고

    에이컨 끄기( 메소드/행위 )정의
    TV 끄기( 메소드 / 행위 ) 정의
    냉장고 끄기( 메소드 / 행위 ) 정의
    리모콘 끄기( 정의X  ) : 리모콘에서 정의을 하면 에어컨 , tv , 냉장고가 동일한 행위을 하기 때문에 정의는 하지 않습니다.
