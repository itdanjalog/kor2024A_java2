- 자바의 타입 종류
    -1. 기본타입(8가지)
        boolean     1바이트    true or false
        char        2바이트    '' 감싼 문자 1개 , 유니코드
        byte        1바이트    -128 ~ 127
        short       2바이트    +-3만 정도
        int         4바이트    +- 21억 정도
        long        8바이트    +- 21억 이상
        float       4바이트    소수점 7자리 표현 , 부동소수점( 정확도가 떨어짐 )
        double      8바이트    소수점 16자리 표현 , 부동소수점 ( 정확도가 떨어짐 )
            -> 정확도가 떨어짐으로 실수형타입이 아닌 문자열 또는 배열 을 이용하여 사용합니다.
    -2. 참조타입
        기본타입 외 모두 : 클래스 , 인터페이스 , 배열[] , 열거타입
        많이 사용되는 타입 : String , List  ,  int[]  ,  -
- (다형성) 타입 변환
    - 1. 자동 / 묵시적 타입 , 작은 바이트에서 큰 바이트
        byte -> short -> int -> long -> float -> double
    - 2. 강제 / 명시적 타입 , 큰 바이트에서 작은 바이트
        byte <- short <- int <- long <- float <- double
